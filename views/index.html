server.js

This file sets up a GraphQL API server using Express and provides an endpoint for interacting with GraphQL. The API exposes a query to validate email addresses. Usage: - To make a GET request, query the `/graphql` endpoint with a URL-encoded query string. - To make a POST request, send a JSON object containing the query to the `/graphql` endpoint.
